1. 발열 CPX는 어떻게 진행되는가

1.1 시험 진행의 큰 흐름 (12분 단일 케이스)

KMLE CPX는 문항당 12분 안에서 단일 환자 케이스를 다룹니다. 발열 케이스의 전형적 흐름은 다음과 같습니다.

입실·라포 형성 : 인사, 자기소개, 손소독

병력청취 : OLD CARTS 체계 + 계통별 동반증상 + 감염 위험요인(여행·접촉·야외활동)

신체진찰 : 활력징후 확인, 의심 계통 중심 진찰. SP가 일부만 실연하고 나머지 소견은 구두/카드로 제공

추정진단·검사계획 설명 : 가장 의심되는 질환 + 감별진단 + 검사 계획

환자교육·마무리 : 응급상황 교육(탈수), 감염병 예방 교육, 질문 받기

| 시험 관련 핵심 코멘트 (반드시 반영) • 발열은 전신 증상이므로 감염 / 비감염으로 나눈 뒤 계통별로 하나하나 물어 감별 합니다. • 환자가 발열을 호소하면 '어디서, 어떻게 체온을 쟀는지' 반드시 확인 합니다. • 최근 여행력 을 묻는 것을 잊지 않습니다. 쯔쯔가무시 의심 시 물린 곳(가피)을 시진합니다. • 발열에는 탈수 증상(어지럼·두근거림·의식 변화) 이 동반될 수 있어 응급상황 교육이 필요합니다. |

| 환자교육 | 잘함/보통/미흡 | 추정진단·법정감염병·진단계획·치료계획·응급/예방교육 |

| PPI(환자-의사관계) | 잘함/보통/미흡 | 쉬운 용어·경청·공감·눈맞춤·신뢰감 |



6.2 항목 → 이중 변환 예시

| 채점 항목(원문) | ① 행동 규칙(환자) | ② 평가 기준(채점) |

| 발열 양상을 물었다 | 양상 질문 시에만 오한·해열제 반응 공개 | 양상 관련 질문 1회+ → '예' |

| 최근 여행/접촉/음식 | 물으면 '없다'고 명확히 부인 | 여행·접촉·음식 중 질문 → '예' |

| 체온 측정 방법 확인 | 어떻게 쟀는지 물어야 38도대 공개 | 측정 부위/방법 질문 → '예' |

| CVA 압통 확인(진찰) | 두드림 명령 시 '(+)' 반환 | CVA 진찰 수행 → 잘함 |

| 공감 표현(PPI) | 공감 발화 전까지 불안 간접 표출 | 공감·경청 적절 → 잘함 |

검사 안내 : CBC·소변검사·소변배양·혈액배양, 필요 시 영상

감염병 예방 교육 : 손씻기, 유행지역 방문 시 예방접종, 체온계로 정확히 측정

7.2 윤리/법적 고려 + 필수술기

| 법정감염병 : 발열 원인이 법정감염병으로 의심되면 종류를 알고, 격리·신고 등의 조치가 필요함을 안내(환자교육 채점 항목). 필수 임상술기 : '혈액배양을 위한 채혈'을 설명하고 시행. 에이전트에서는 술기 설명에 대한 환자 반응(동의·불편 호소)으로 구현. |
