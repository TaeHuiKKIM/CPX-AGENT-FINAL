# 2026 의료 AI·SW 융합 해커톤 발표 기획서 — Yes/No 100점 채점 반영판

## 발표 핵심 메시지

> “저희 서비스는 AI 표준화환자와 대화한 뒤, 발열 채점표 예시를 기준으로 40개 항목을 Yes/No로 자동 채점하고, 누락된 질문과 진찰을 이유와 함께 알려주는 CPX 학습 플랫폼입니다.”


## 채점 기준 요약

- 기준: 발열 채점표 예시
- 방식: 40개 항목 전체 Yes/No
- 총점: `Yes 개수 / 40 × 100`
- 가중치: 없음
- 부분점수: 없음
- 기존 `잘함/보통/미흡` 표기는 사용하지 않음


## 1. Problem

의대생은 실제 표준화환자와 충분히 연습하기 어렵다. 기존 AI CPX 서비스는 단순 키워드 채점에 머물거나, 환자가 묻지도 않은 정보를 먼저 말해 CPX의 핵심인 능동 문진 능력을 제대로 평가하지 못한다.

## 2. Solution

### 2.1 AI 표준화환자

AI 환자는 주소만 먼저 말하고, 학생이 묻기 전까지 과거력, 가족력, 사회력, 약물력, 동반증상을 누설하지 않는다.

### 2.2 발열 채점표 기반 자동 채점

- 발열 채점표 예시의 40개 항목을 그대로 사용한다.
- 모든 항목은 Yes/No로만 채점한다.
- 최종 점수는 `Yes 개수 / 40 × 100`으로 계산한다.
- 항목별 가중치와 부분점수는 없다.

### 2.3 이유가 있는 피드백

단순히 “여행력 미확인”이라고만 보여주지 않고, “말라리아, 장티푸스 등 감염성 발열 감별을 위해 필요했다”처럼 이유를 함께 설명한다.

## 3. Scale-up

발열 8케이스의 환자 데이터는 케이스마다 다르지만, 채점 엔진은 동일한 40개 Yes/No 기준을 사용한다. 이후 기침, 복통, 흉통 등 다른 주제로 확장할 때도 먼저 해당 주제의 채점표를 구조화한 뒤 동일한 방식으로 자동 채점할 수 있다.

## 4. Development

- 실시간 STT-LLM-TTS 기반 음성 대화
- AI 환자 정보공개 가드레일
- Transcript 기반 40항목 Yes/No 자동 채점
- 근거 발화 하이라이트
- No 항목 중심의 개인화 피드백 생성

## 5. 한 줄 비교표

| 구분 | 기존 서비스 | 우리 서비스 |
|---|---|---|
| 환자 응답 | 정보 누설 가능 | 묻기 전 정보 제공 금지 |
| 채점 방식 | 키워드 중심 | 발열 채점표 40항목 Yes/No |
| 점수 산식 | 불명확하거나 가중치 기반 | Yes 개수 / 40 × 100 |
| 피드백 | 누락 항목 나열 | 누락 이유와 보완 방향 제공 |
| 확장 | 케이스 수 중심 | 채점표 구조화 기반 확장 |
