# CPX 평가 항목 분석 보고서 — 발열 Yes/No 100점 적용판

## 1. 최종 적용 기준

타사 AI CPX 피드백 화면과 PMR 계열 평가 항목은 참고 자료로만 사용한다. 실제 구현 기준은 **발열 채점표 예시의 40개 항목 Yes/No 자동 채점**으로 통일한다.


---

## 공통 채점 기준 v2 — 발열 채점표 예시 기반 Yes/No 100점

본 프로젝트의 자동 채점은 **발열 채점표 예시**를 기준으로 통일한다. 기존의 `잘함/보통/미흡`, 항목별 가중치, PMR 다축 가중 평가는 제거하고, 모든 항목을 **Yes / No**로만 판단한다.

### 점수 산식

```text
총 항목 수 = 40개
Yes = 1, No = 0
최종 점수 = (Yes 개수 / 40) × 100
항목당 점수 = 2.5점
```

- 별도 가중치는 두지 않는다.
- 부분점수는 두지 않는다.
- 학생이 같은 의미의 질문·진찰·설명을 명확히 수행하면 `Yes`로 인정한다.
- 환자가 먼저 말한 내용을 학생이 능동적으로 확인하지 않은 경우는 원칙적으로 `No`로 처리한다.
- 최종 피드백은 점수보다 `No` 항목을 중심으로 “무엇을 놓쳤는지”와 “왜 필요한지”를 설명한다.

## 4. 피드백 구조

피드백은 3단계로 제공한다.

1. **총점 요약**: Yes 개수, No 개수, 100점 환산 점수.
2. **누락 항목 요약**: No 항목을 병력청취, 신체진찰, 교육, PPI로 묶어 제시.
3. **개선 가이드**: 각 No 항목에 대해 왜 필요한지와 다음 실습에서 어떤 문장을 쓰면 되는지 제시.

## 5. 예시 피드백

```text
총점은 72.5점입니다. 총 40개 항목 중 29개를 수행했습니다.
병력청취에서는 발열 양상과 전신 증상 확인은 좋았지만, 최근 여행력·접촉력·음식 복용을 확인하지 않았습니다.
발열 환자에서는 말라리아, 장티푸스, 식중독 등 감염성 원인을 배제하기 위해 역학력을 반드시 확인해야 합니다.
```
