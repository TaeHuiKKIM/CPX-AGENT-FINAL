# CPX 발열 8케이스 최종본 — Yes/No 100점 채점 반영판

## 공통 적용 사항

아래 8개 발열 케이스의 환자 정보, 문진, 진찰, 감별진단, 교육 내용은 유지하되, 최종 자동 채점 방식은 전부 **발열 채점표 예시 기반 40개 Yes/No 항목**으로 통일한다.


## 채점 기준 요약

- 기준: 발열 채점표 예시
- 방식: 40개 항목 전체 Yes/No
- 총점: `Yes 개수 / 40 × 100`
- 가중치: 없음
- 부분점수: 없음
- 기존 `잘함/보통/미흡` 표기는 사용하지 않음
