| CPX 평가 항목 분석 보고서 타사 AI CPX 피드백 화면 기반 — 어떤 역량을 평가하는지 구조화 |



| 01  교수 총평 (Examiner Summary)    서술형 피드백 |



| 전반적 수행 총평 | AI가 수험생의 전체 진료 흐름을 종합 평가하여 서술형으로 피드백. 강점과 약점을 함께 기술하며, 임상 추론의 논리성·체계성을 중심으로 코멘트. |



| 잘한 점 / 개선할 점 | 카드 형태로 긍정적 수행 요소(잘한 점)와 보완 필요 영역(개선할 점)을 분리 제시. 단순 점수가 아닌 행동 기반 피드백. |



| 02  미국·영어권 표준지표 (PMR) 평가    6개 항목 |



| 병력 기술 | 현병력 청취 능력. 주증상, 발병 시기, 경과, 악화·완화 인자, 관련 증상 탐색 여부 평가. |



| 환자 안심 및 이해 | 환자의 감정·불안을 인식하고 적절히 반응했는지, 설명의 명확성 및 공감적 소통 평가. |



| 환자 정보 | 과거력(PMH), 가족력, 사회력(흡연·음주·직업 등) 수집의 완성도 평가. |



| 감별 진단 | 임상 정보에 근거한 합리적 DDx 나열 및 우선순위 설정 능력 평가. |



| 진단 계획 (검사) | DDx 검증을 위한 적절한 검사(lab, imaging 등) 선택 및 우선순위 판단 평가. |



| 근거 기술 (EBG) | 진단·치료 결정의 근거를 명확히 제시했는지, 임상 가이드라인·증거 기반 의학 적용 여부 평가. |



| 03  사례별 체크리스트 — 병력 청취 항목    증상 특이적 |



| 주증상 onset·duration·character·location·radiation·aggravating/relieving factors·severity 체계적 탐색 (OLDCARTS 프레임워크) |

| Associated symptoms 탐색 — 주증상과 연관된 동반 증상 망라 |

| Review of Systems (ROS) — 장기 계통별 증상 확인 |

| 과거력(PMH), 약물력, 알레르기, 가족력, 사회력 각각 독립 항목으로 체크 |

| 기능 상태(ADL, exercise tolerance) 및 직업·환경 노출 등 맥락 정보 수집 |

| 소아과 케이스의 경우 발달력, 예방접종, 출생력 추가 항목 포함 |



| 04  사례별 체크리스트 — 신체 검진 항목    계통별 |



| 생체징후(Vital signs) 확인 요청 여부 |

| 계통별 신체 검진 항목 누락 없이 수행 여부 (심폐, 복부, 신경학적 등) |

| 증례 핵심 병리에 특화된 검진 기술 (예: Murphy's sign, Kernig sign 등) |

| 검진 시 환자 설명 및 동의 절차 |



| 05  감별 진단 및 교육 계획    임상 추론 |



| Diagnosis category | 제시한 진단명이 증례 핵심 질환 카테고리에 속하는지 여부 (예: Respiratory, Endocrine 등 대분류 정확성). |



| Diagnosis (세부) | 최종 또는 1순위 진단의 정확도. 근거(병력, 이학적 소견, 검사)를 논리적으로 연결했는지 평가. |



| Drop Pitfall Dx | DDx에서 놓치면 안 되는 위험 질환(must-not-miss)을 포함했는지. 증례의 Red flag에 민감하게 반응했는지 여부. |



| Other inflammatory conditions | 증례와 연관된 염증성 질환 감별 포함 여부. |



| Infection / Neoplastic / Fungal / Parasites | 감염성·종양성·진균·기생충 병인을 DDx에 적절히 포함했는지 평가. |



| 06  검사 선택 및 임상 추론    Evidence-based |



| CBC, CMP, inflammatory markers (CRP, ESR), tumor markers 등 기본 lab 선택의 적절성 |

| Biopsy, Colonoscopy 등 침습적 검사 적응증 판단 능력 |

| CT scan of abdomen/pelvis — 적응증 인식 및 시행 시점 판단 (소견 의심 시 제안 여부) |

| 검사 우선순위 설정 (단계적 검사 전략 vs 무분별한 ordering) |

| 검사 결과 해석 및 이를 통한 진단 재확인 과정 |



| 07  환자 교육 및 상담    Communication |



| 진단명 및 병태생리를 환자 수준에서 이해 가능하게 설명하는 능력 |

| 치료 계획(약물, 생활습관, 추적 관찰 등) 전달의 명확성 |

| 예후 및 합병증에 대한 균형 잡힌 설명 (불필요한 불안 조성 없이) |

| 다음 방문 시점, 응급 증상 발생 시 행동 지침 안내 |



| 08  추가 고득점 항목 (Star items)    고난도 |



| Obscure / atypical presentation 인식 — 전형적이지 않은 증상 패턴에서 정확한 진단 도달 |

| Systemic thinking — 국소 증상에서 전신 질환으로 사고 확장 |

| 공존 질환(comorbidity) 고려 및 이로 인한 진단·치료 계획 수정 |

| 환자 선호도·가치 반영 (shared decision making 요소) |



| 총 정리  이 CPX는 크게 ① 임상 추론 (병력→DDx→검사 선택→진단), ② 의사소통 (환자 안심·교육·공감), ③ 체크리스트 완성도 (계통별 누락 여부), ④ 근거 기반 의학 (EBG, 가이드라인 적용) 4개 축으로 역량을 평가함. 단순 지식이 아닌 임상 현장에서의 실행력을 AI가 구조화하여 점수화하는 방식. |
